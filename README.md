 自动寻球小车

项目简介：

本项目设计并实现了一款基于 OpenMV H7 Plus + STM32F103RCT6 双芯片架构的自动寻球小车，支持自动寻球与蓝牙遥控双模式，可自主识别并抓取网球，用于解决网球场频繁捡球的重复性劳动问题。

项目融合了机器视觉、PID控制、双芯片串口通信、舵机时序控制、无线遥控等嵌入式核心技术，具备较强的可移植性，更换识别算法后可适配乒乓球、羽毛球等其他球类。


核心功能：

- 自动模式：OpenMV识别网球 -> PID平滑追踪 -> 到达目标 -> 触发STM32执行抓取动作 -> 放入储球区
- 遥控模式：通过手机蓝牙APP控制小车前进/后退/左转/右转/抓取/停止
- 模式切换：遥控模式按'7'进入自动，自动模式按'6'回到遥控


技术栈：

- 视觉主控：OpenMV H7 Plus（MicroPython）
- 执行控制：STM32F103RCT6（C语言，Keil MDK）
- 视觉识别：颜色阈值分割 + 圆形检测 + 最大色块追踪
- 运动控制：双闭环PID控制器（方向 + 距离）
- 双芯片通信：UART串口（9600bps）
- 无线遥控：HC-08蓝牙模块 + 手机APP
- 电机驱动：TB6612FNG + N20直流减速电机
- 舵机控制：MG996R舵机 + 金属机械爪
- 电源管理：双电池独立供电（11.1V + 7.4V）

硬件连接：

OpenMV引脚分配

| 功能 | 引脚 | 说明 |
|------|------|------|
| 左轮PWM | P7 | 定时器4通道1 |
| 右轮PWM | P8 | 定时器4通道2 |
| 左轮方向A | P6 | AIN1 |
| 左轮方向B | P9 | AIN2 |
| 右轮方向A | P2 | BIN1 |
| 右轮方向B | P3 | BIN2 |
| STM32串口TX | P4 (UART3 TX) | 发送'1'触发抓取 |
| STM32串口RX | P5 (UART3 RX) | 预留接收STM32反馈 |
| 蓝牙串口RX | P0 (UART1 RX) | 接收手机指令 |
| 蓝牙串口TX | P1 (UART1 TX) | 预留发送状态 |

注意：OpenMV与STM32的UART需交叉连接（TX->RX, RX->TX），并共地。

双芯片通信协议：

| 方向 | 指令 | 含义 |
|------|------|------|
| OpenMV -> STM32 | '1' | 触发抓取动作 |
| STM32 -> OpenMV | 'D'（预留） | 抓取完成确认 |

蓝牙遥控指令：

| 指令 | 编码 | 功能 |
|------|------|------|
| '1' | 49 | 前进 |
| '2' | 50 | 左转 |
| '3' | 51 | 右转 |
| '4' | 52 | 后退 |
| '5' | 53 | 抓取 |
| '6' | 54 | 切换到自动模式 |
| '7' | 55 | 停止/切换回遥控模式 |


核心算法：

双闭环PID控制

小车通过两个独立的PID控制器实现平滑追踪：

- 方向PID（x_pid）：控制目标在图像中的水平偏移量，输出左右轮差速
- 距离PID（h_pid）：控制目标面积与预设阈值的偏差，输出前进/减速速度

参数配置：
- x_pid = PID(p=0.5, i=0.01, imax=50)
- h_pid = PID(p=0.05, i=0.01, imax=30)

抗干扰策略

- 抗积分饱和：积分项钳位在±imax范围内
- 微分低通滤波：一阶RC滤波（截止频率20Hz），抑制视觉噪声放大
- 抓取防误触：连续稳定55帧（约3.6秒）才触发抓取
- PID输出限幅：±100，防止电机过驱

颜色识别

使用LAB颜色空间阈值分割绿色网球，通过find_blobs() + find_max()定位面积最大的目标。若视野中无网球，小车原地旋转搜索。


演示效果

- 完整闭环：识别 -> 追踪 -> 抓取 -> 储存
- 连续抓取3个网球，成功率100%
- PID控制平滑，无振荡
